COMANDA AZUL - BAR AZUL

Projeto feito em HTML, CSS e JavaScript puro.
Não usa banco de dados e não precisa de instalação.

COMO USAR:
1. Abra o arquivo index.html no navegador.
2. Use as mesas, balcão e extras para adicionar produtos.
3. Use a busca dentro da comanda para localizar produtos.
4. Ao enviar uma comanda, escolha a forma de pagamento.
5. Se a forma for Dinheiro, informe o valor recebido; o troco é calculado automaticamente.
6. O Valor do Dia oculta os valores na tela por segurança.
7. Use Editar no Valor do Dia para corrigir comanda, itens e pagamento em um pop-up.
8. Ao clicar em Concluir e gerar PDF, o sistema baixa um relatório com todos os valores, reseta o dia, limpa as comandas e remove todos os extras.

OBSERVAÇÃO:
Os dados ficam salvos temporariamente no navegador por localStorage.
Para apagar tudo manualmente, limpe os dados do site no navegador.
