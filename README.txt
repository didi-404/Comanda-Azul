COMANDA AZUL - versão corrigida

Como usar:
1. Abra o arquivo index.html no navegador.
2. Use os botões + e - para adicionar/remover produtos.
3. Para mudar o nome de uma comanda, edite diretamente o campo do nome e aperte Enter, ou clique em Renomear.
4. O botão "Voltar padrão" devolve o nome original daquela comanda.
5. O botão "Remover este Extra" aparece apenas nos containers Extra.
6. O botão "Concluir" mostra o total vendido e faz reset automático:
   - zera o Valor do dia;
   - remove todas as Extras;
   - limpa produtos das comandas;
   - retorna os nomes para Mesa 01, Mesa 02, Mesa 03, Mesa 04 e Balcão.

Feito apenas com HTML, CSS e JavaScript puro. Não usa banco de dados.
